### lazy git

Add below code into your ~/.bash_profile(Mac) or ~/.bashrc(Linux)

{% highlight shell %}
# lazy git
function lgit() {
    git add .
    git commit --allow-empty-message -a -m "$1"
    git push
}
{% endhighlight %}
